description: Start Eleventy dev server with live reload
---

Start the Eleventy development server with auto-rebuild and live reload.

This command:
1. Starts dev server on http://localhost:8080
2. Watches `src/` for file changes
3. Auto-rebuilds on any change
4. Live reloads browser automatically

**When to use:** During active development - you can edit `src/` files and see changes instantly without manual builds.

**Equivalent to:** `npm start`

**Note:** Server runs in background. Visit http://localhost:8080 in your browser.
